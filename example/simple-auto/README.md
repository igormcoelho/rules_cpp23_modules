# Example: simple-auto

This is an example for project [rules_cpp23_modules](https://github.com/igormcoelho/rules_cpp23_modules).

> IMPORTANT: This is for Bazel 8 and before, since after Bazel 9 there's official support for modules (at least for clang!).

> For Bazel 9 with clang, see [demo9/simple](../../demo9/simple/README.md).

### Description

This is very similar to the  [demo/simple](../../demo/simple/README.md),
but this one compiles the CMI automatically! No manual work! =)

Try this for GCC!
